This plugin extends WooCommerce by setting extra custom fields for every simple product.

== Description ==

This plugin extends WooCommerce by allowing some additional custom product meta-data to single product. This product meta-data can be carried to single product page to checkout and even can be email to the customer.

Features :

1. Add custom fields to single product template.
2. The name of the fields are completely customisable. 
3. User is flexible to choose whether to display them in single product page or not. 
4. User is flexible to choose whether to save the custom field data for admin purpose at checking out. 
5. User is flexible to choose whether to mail the custom field data to customer at checking out. 
6. It does not create any additional database/tables. So, it does not consume much space.

== Installation ==

Download the "Custom Products fields woo" plugin . Extract it. Then copy and paste it to your 
wp-content/plugins/ directory. Then go to your wordpress admin account and activate it.
